① 生成密钥（web 流程 · 免 author 工具 token）

GitHub 登录后，填加密块 id 与允许解锁的 GitHub 用户名，点按即由服务端生成密钥并写入 KV，返回密钥 JSON。把该 JSON 交给 author.mjs --key-json 本地加密即可，无需 admin token。注意：服务端会回传明文密钥（仅经 HTTPS），请妥善保存该 JSON。
