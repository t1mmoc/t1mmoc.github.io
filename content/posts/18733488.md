```
build: {
    rollupOptions: {
      output: {
        entryFileNames: `assets/[name].js`, // 打包后的入口文件名
        chunkFileNames: `assets/[name].js`, // 打包后的代码块文件名
        assetFileNames: `assets/[name].[ext]`, // 打包后的静态资源文件名
      },
    },
  },
```

增加这段代码即可，加的位置需要注意。
