# 直接开始

3. 配置 Apache  
   编辑 httpd.conf 文件：  
   打开 WampServer 安装目录下的 bin/apache/apache版本号/conf/httpd.conf 文件。  
   取消以下行的注释（删除行首的 #）：

```
LoadModule ssl_module modules/mod_ssl.so
Include conf/extra/httpd-ssl.conf
LoadModule socache_shmcb_module modules/mod_socache_shmcb.so
```

编辑 httpd-ssl.conf 文件：  
打开 conf/extra/httpd-ssl.conf 文件。  
修改以下参数以指向您生成的证书文件：

```
SSLCertificateFile "${SRVROOT}/conf/server.crt"
SSLCertificateKeyFile "${SRVROOT}/conf/server.key"
```

![image](/images/18765053/3336551-20250311150042063-2078400234.jpg)  
放的位置是这里。  
确保 DocumentRoot 和 ServerName 设置为您网站的根目录和域名（如 localhost）。  
4. 重启 WampServer  
完成上述配置后，重启 WampServer 以使更改生效。
