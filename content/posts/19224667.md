# 用来留给自己的，避免每次都连那半天转不上的github

`export PS1="$PS1\[\e]1337;CurrentDir="'$(pwd)\a\]'`

`precmd () { echo -n "\x1b]1337;CurrentDir=$(pwd)\x07" }`

```
    echo -en "\e]1337;CurrentDir=$PWD\x7"
end
```

一般就第一条就行。
