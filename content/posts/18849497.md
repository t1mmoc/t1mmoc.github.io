# 前言

经常做日程管理的朋友们都知道，日程管理不是什么好活  
最近我也是弄起来这个东西了。

# 要求

- 能够支持手机与电脑同步日程管理
- 能够使用原生手机日历的提醒
- 电脑同样能够有一点提醒，最好也用Windows原生的提醒。

# 寻觅

先找了一个`企业微信`和`钉钉`，这俩都差不多，没怎么碰，因为企业因素比较高，而且只能从系统日历同步到他们自己的日历不符合我使用系统原生日历的需要。  
**注：两个软件都给了`ex方法`同步到系统日历，但是这样和邮箱提供的就没有区别了**

# 发现

接下来找邮箱，`gmail`首先排除，墙壁的原因。  
`outlook`是我第一个考虑的，毕竟微软出品，而且**有逼格**，而且据说非常注重日程管理，邮箱似乎还没有他日程管理的重要。

### outlook之旅

**缺点：**

- 同样是墙壁原因，但这是弱墙，下载超慢，还是能连，概率断线
- 软件启动速度超慢
- 别用qq.com直接注册微软账号，不然有的你哭，现在还在用qq.com的，趁着没买东西赶紧去重新注册一个outlook.com，微软的兼容性真的没话说，如兼。
- 用outlook.com注册的新微软账号被限制使用了两次，两次都是需要我用手机重新验证，还好不像腾讯那样要签莫须有的保证书

**优点：**

- 有逼格
- ui好看
- 支持系统原生通知
- 没了。

对了，outlook同时是邮箱还是应用，所以我把他应用的缺点也列在上面了。  
如果用outlook应用去使用其他邮箱，不好意思，不支持。

### 网易163邮箱 vs QQ邮箱

好消息，这俩都一样。坏消息，163邮箱那边想接受一个日程是不行的，也可能是只能接受来自163的邮箱日程，有看到日程，但是没有接受按钮，非常无语。  
在两个一样的情况下，163有一个缺点，当然选QQ了。（163的付费点实在是太多了）  
什么你的意思是QQ那一长串的不好看？虽然QQ的邮箱已经没办法取英文名了只有foxmail名  
但是微信还没删啊，可以用微信注册qq邮箱然后取英文id，敲两下左上角的qq不比163好打。

好，选定了QQ邮箱作为日程同步方案，接下来是选择手机、电脑端应用。

# QQ邮箱之旅

手机端其实不需要任何软件，手机自带ex账号登录，这里不列，懒。
