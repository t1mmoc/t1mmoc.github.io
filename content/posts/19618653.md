## der转.0证书

```
# 步骤1: 将DER格式转换为PEM格式
openssl x509 -inform DER -in cacert.der -out cacert.pem

# 步骤2: 计算证书的旧版subject hash（Android系统使用）
openssl x509 -inform PEM -subject_hash_old -in cacert.pem | head -1
# 输出示例: 9a5ba575

# 步骤3: 将PEM证书重命名为 <hash>.0 格式
mv cacert.pem 9a5ba575.0
```

# 安装证书

```
adb root
adb shell mount -o rw,remount /system
adb push 9a5ba575.0 /system/etc/security/cacerts/
adb shell chmod 644 /system/etc/security/cacerts/9a5ba575.0
```

如果你也和我一样push了以后找不到，那就在shell里面

```
mount -o rw,remount /system
cp /storage/emulated/0/9a5ba575.0 /etc/security/cacerts/
ls /etc/security/cacerts/9a5ba575.0
chmod 644 /etc/security/cacerts/9a5ba575.0
```

记得root权限

# 设置代理

```
adb.exe shell settings put global http_proxy 10.0.2.2:8080
```

清除代理：

```
adb.exe shell settings delete global http_proxy
```
